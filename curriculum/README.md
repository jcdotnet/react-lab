# 📄 Curriculum Vitae

My professional CV, built to practice React and integrated into this laboratory in 2026 as a legacy modernization exercise.

🚀 **[Live Demo](https://react-lab-ashy.vercel.app/)**

### Technical Highlights:
*   **Legacy Migration**: Successfully migrated from Create React App (CRA) to **Vite** for faster builds and modern HMR.
*   **Repo Architecture**: Integrated into the `react-lab` monorepo using `git subtree` to preserve original commit history (2016-2025).
*   **Data Refactor**: Decoupled from a legacy Laravel API; implemented a local, structured data model for high reliability and zero external dependencies.
*   **CI/CD**: Automated deployment pipeline configured via **Vercel** with custom root directory routing.
