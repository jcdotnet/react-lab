# Curriculum Vitae

My professional CV, built to practice React and integrated into this laboratory in 2026 for better organization.

This project was initially created with Create React App, fetched CV data from this [Laravel API](https://github.com/jcdotnet/php-lab/blob/main/laravel/personal-site/routes/api.php) and was deployed on josecarlosroman.com. It now uses Vite, is decoupled from the API and is automatically deployed via Vercel.

**[Live Demo](https://react-lab-ashy.vercel.app/)**
